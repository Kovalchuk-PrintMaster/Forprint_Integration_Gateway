# Gateway v0.6 Backward Compatibility Gates

Gateway v0.6 adds backward compatibility gates to protect accepted v0.3, v0.4
and v0.5 concepts.

The gates protect:

```text
v0.3 channel intake examples
v0.4 adapter readiness
v0.5 compatibility matrix
v0.5 replay fixtures
canonical route target module ids
no live delivery
Accounting/1C/posting blocks
Mobile App planned/future status

Breaking changes require Blueprint approval.
